# AI-Driven Quant Trading Dashboard

An end-to-end Machine Learning project that optimizes stock portfolios and predicts future price trends.

##  Progress
- **Day 1:** Built a real-time data pipeline using `yfinance`.
- *.*Day 2:** Implemented Markowitz Efficient Frontier for portfolio optimization.
- **Day 3:** Designed a Deep Learning LSTM architecture using **PyTorch**.
- **Day 4:** Developed the training loop and saved model weights.

##  Tech Stack
- **Language:** Python 3.12
- **AI Framework:** PyTorch
- **Finance:** Pandas, NumPy, yfinance
